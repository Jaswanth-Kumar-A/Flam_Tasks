## Setup Instructions

1. Clone the repository:
   ```bash
   git clone <repository-url>
   cd hr-dashboard-advanced
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Build for production:
   ```bash
   npm run build
   ```
